#### go_gin_web 练习

~~~~

git remote add origin git@github.com:jierui303-byte/go_gin_web

~~~~
#Gin包安装命令：
`go get -u github.com/gin-gonic/gin`

`go get gopkg.in/mgo.v2`
~~~~
